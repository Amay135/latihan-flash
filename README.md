# Latihan Flask

Repositori ini berisi proyek-proyek kecil dan latihan dasar saya dalam mempelajari framework **Flask** menggunakan bahasa pemrograman **Python**.

## Fitur Utama
* Memahami konsep routing dasar.
* Implementasi template Jinja2.
* Pengelolaan form dan request data.
* (Tambahkan fitur lain jika ada, contoh: Integrasi database SQLite).

## Persyaratan (Prerequisites)
Sebelum menjalankan aplikasi, pastikan Anda telah menginstal:
* [Python 3.x](https://www.python.org/)
* [Flask](https://flask.palletsprojects.com/)

## ⚙️ Cara Menjalankan
1. Clone repositori ini:
   ```bash
   git clone [https://github.com/Amay135/latihan-flash.git](https://github.com/Amay135/latihan-flash.git)
``
2. Masuk ke direktori proyek:
  ```
  cd latihan-flash
  ```
3. Buat virtual environment (opsional tapi disarankan):
   ```
   python -m venv venv
   source venv/bin/activate  # Untuk Windows: venv\Scripts\activate
4. Instal dependencies:
   ```
   pip install -r requirements.txt
5. Jalankan aplikasi:
   ```
   python app.py
